🎓 Projeto Educa Plus: Gestão de AprendizagemEste repositório é dedicado ao desenvolvimento técnico do sistema Educa Plus. O diferencial deste projeto é que ele está sendo construído a partir da documentação de requisitos elaborada por um colega, servindo como um exercício prático de validação de requisitos e implementação fiel às regras de negócio estabelecidas.  📄 Documentação de ReferênciaO planejamento original (Documento de Visão e Especificação de Requisitos), que servirá como base para este desenvolvimento, pode ser consultado aqui:👉 EducaPLUS.pdf🎯 Objetivos de AprendizadoDiferente de um projeto pessoal comum, este repositório foca em:Validação de Requisitos: Aplicar na prática a implementação de Requisitos Funcionais (RF01 a RF22) e Não Funcionais (Segurança, Desempenho e Usabilidade) descritos na documentação.  Fidelidade ao Escopo: Desenvolver as funcionalidades de gestão acadêmica, avaliação, comunicação e AVA conforme planejado.  Padrões de Engenharia: Seguir as restrições de arquitetura (MVC/Clean Architecture) e segurança (RBAC e LGPD) especificadas.  🛠️ Tecnologias de ImplementaçãoComo meu objetivo é me tornar uma desenvolvedora Full Stack, utilizarei:Backend: Java (Linguagem principal para lógica de negócio).Banco de Dados: PostgreSQL (Persistência e integridade dos dados).Frontend: JavaScript, HTML e CSS (Interface responsiva).Este projeto utiliza a documentação cedida por Matheus Senna Mascarenhas como base de desenvolvimento acadêmico na UCSAL.
