Este software foi desenvolvido como um desafio acadêmico para a disciplina de Engenharia de Requisitos.

O projeto consistiu em implementar um sistema funcional baseando-se inteiramente na documentação técnica elaborada pelo meu colega Matheus Sena. O meu foco foi o aprendizado prático de transformar requisitos e regras de negócio em um código estruturado e integrado.

Principais aprendizados:

    Tradução de documentação técnica para implementação de backend e frontend.

    Modelagem e estruturação de banco de dados relacional.

    Integração entre camadas de aplicação.
